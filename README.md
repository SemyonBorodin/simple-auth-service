# Authentication Service

## Описание

На данный момент реализован микросервис (auth-service) для аутентификации и регистрации пользователей. Включает следующие функции:
- Регистрация пользователей
- Аутентификация через JWT
- Обновление токенов

## Стек технологий
- Spring Boot
- Hibernate
- JWT
- PostgreSQL
